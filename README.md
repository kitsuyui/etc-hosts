# etc-hosts

Generate and Install secure `/etc/hosts` file.

Currently tested on macOS v10.12.1 Sierra.
I'm planning to adapt for other OS.

## Usage

```console
$ make
$ sudo make install
```

## Requirements

- make(1)
- [ninja.build](https://ninja-build.org/)
- [curl](https://curl.haxx.se/)
